# ICT交易策略详解

## 什么是ICT？

ICT (Inner Circle Trader) 是一种机构级交易方法，核心思想是**跟随机构资金的流向**，识别机构建仓、出货的痕迹。

## 核心概念

### 1. Order Block (OB) - 订单区块

**定义**: 机构建仓或平仓的区域，通常表现为放量长影线后的盘整。

**特征**:
- 成交量显著放大
- 长上/下影线
- 随后价格离开该区域

**类型**:
- **看涨OB**: 机构买入区域，通常是下跌后的绿色K线
- **看跌OB**: 机构卖出区域，通常是上涨后的红色K线

**识别规则**:
```
1. 找到最近的趋势反转点
2. 确认该位置有明显的放量
3. 确认该位置有长影线
4. 随后价格形成盘整区间
5. 该区间即为OB
```

### 2. Fair Value Gap (FVG) - 公平价值缺口

**定义**: 三根连续K线中，中间一根的最低价比前一根最高价高，或中间一根的最高价比前一根最低价低。

**类型**:
- **看涨FVG**: (Low[1] > High[2]) 且价格回测到该区域时是买入机会
- **看跌FVG**: (High[1] < Low[2]) 且价格回测到该区域时是卖出机会

**FVG公式**:
```
看涨FVG: Low[i-1] > High[i+1]
看跌FVG: High[i-1] < Low[i+1]
```

### 3. Liquidity (流动性池)

**定义**: 止损单集中的区域，通常是前期高点/低点。

**类型**:
- ** highs (上方的流动性池)
- **Lows (下方的流动性池)

**交易逻辑**:
```
1. 价格向流动性池移动
2. 触及流动性池（止损猎杀）
3. 价格反转
4. 顺势交易
```

### 4. Market Structure Shift (MSS) - 市场结构转换

**定义**: 趋势改变的信号，高点被突破或低点被跌破。

**看涨MSS**:
```
1. 价格创出更高高点(HH)
2. 回调不破前低(HL)
3. 突破HH高点
```

**看跌MSS**:
```
1. 价格创出更低低点(LL)
2. 回抽不破前高(LH)
3. 跌破LL低点
```

### 5. Break of Structure (BOS) - 结构突破

**定义**: 趋势中的回调被突破，确认趋势延续。

**看涨BOS**:
```
1. 上升趋势中
2. 价格回调形成低点
3. 突破回调高点
4. 趋势延续
```

## 实战交易流程

### 看多交易

```
步骤1: 确认下跌趋势结束
  - 寻找看涨OB
  - 寻找流动性池被扫

步骤2: 等待MSS确认
  - 突破最近高点
  - 成交量放大

步骤3: 寻找入场点
  - 等待价格回测FVG
  - 在OB上方入场

步骤4: 设置止损
  - 止损放在OB下方
  - 或放在最近低点下方

步骤5: 目标位
  - 下一流动性池
  - 或3:1盈亏比
```

### 看空交易

```
步骤1: 确认上涨趋势结束
  - 寻找看跌OB
  - 等待流动性池被扫

步骤2: 等待MSS确认
  - 跌破最近低点
  - 成交量放大

步骤3: 寻找入场点
  - 等待价格回测FVG
  - 在OB下方做空

步骤4: 设置止损
  - 止损放在OB上方

步骤5: 目标位
  - 下一流动性池
```

## 与其他策略结合

ICT策略可以与以下方法结合：

1. **+ PA价格行为**: 确认K线形态
2. **+ 均线**: 确认支撑阻力
3. **+ 成交量**: 确认机构参与度
4. **+ 斐波那契**: 寻找回撤目标位

## 常见误区

1. **不要在趋势中途逆势交易**: MSS是确认趋势改变的关键
2. **不要忽视成交量**: 机构参与的标志
3. **不要不止损**: 机构也会错，止损是保护
4. **不要过度交易**: 等待高概率信号
