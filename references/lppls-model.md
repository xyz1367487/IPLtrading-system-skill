# LPPLS模型详解

## 什么是LPPLS？

LPPLS (Log-Periodic Power Law Singularity) 模型是一种用于预测金融市场泡沫和崩盘的数学模型。由物理学家D. Sornette提出，基于**自组织临界性**理论。

## 核心公式

```
P(t) = A + B(t_c - t)^m + C(t_c - t)^m * cos(ω*ln(t_c - t) + φ)
```

### 参数解释

| 参数 | 含义 | 典型范围 |
|------|------|----------|
| A | 长期趋势项 | 线性或指数 |
| B | 振幅系数 | < 0 (泡沫) 或 > 0 (崩盘后反弹) |
| C | 振荡幅度 | |
| t | 当前时间 | |
| t_c | 临界时间（预测崩盘点）| 未来某点 |
| m | 稳定性指数 | 0 < m < 1 |
| ω (omega) | 崩溃频率 | 4 < ω < 13 |
| φ (phi) | 相位 | -π < φ < π |

## 模型原理

### 1. 泡沫形成过程

```
正常上涨 → 加速上涨 → 出现LPPLS特征 → 临界点 → 崩盘
```

### 2. 对数周期振荡

模型中的 `cos(ω*ln(t_c - t) + φ)` 项解释了为什么泡沫在崩盘前常表现为:
- 快速上涨
- 间歇性回调
- 波动加剧

### 3. 临界指数

- **m值**: 接近1表示泡沫正在加速，接近0表示趋于稳定
- **ω值**: 高值表示高频振荡，崩盘风险大

## 识别特征

### LPPLS泡沫信号

1. **价格加速上涨**: 呈现指数增长特征
2. **波动率增加**: 振幅越来越大
3. **对数周期振荡**: 出现规律的回调-反弹模式
4. **时间收敛**: 每次反弹间隔越来越短

### 计算LPPLS指标

```python
def calculate_psi(t, tc, m, omega, C):
    """计算LPPLS泡沫指数"""
    dt = tc - t
    if dt <= 0:
        return None
    psi = C * (dt ** m)
    return psi

# 预警阈值
if psi < -0.5:
    alert = "强泡沫预警"
elif psi < -0.3:
    alert = "中度泡沫预警"
elif psi < -0.1:
    alert = "轻度泡沫预警"
```

#### 参数拟合

使用非线性最小二乘法拟合参数:
```python
from scipy.optimize import curve_fit

def lppls(t, A, B, C, tc, m, omega, phi):
    dt = tc - t
    return A + B * (dt ** m) + C * (dt ** m) * np.cos(omega * np.log(dt) + phi)

# 拟合
popt, pcov = curve_fit(lppls, t, price, p0=initial_guess)
```

#### 预警判断

```python
def get_alert_level(psi, m, omega):
    score = 0
    
    # psi指标
    if psi < -0.5: score += 2
    elif psi < -0.3: score += 1
    
    # m值
    if m > 0.8 or m < 0.2: score += 1
    
    # omega值
    if omega > 10: score += 1
    
    if score >= 3:
        return "高风险"
    elif score >= 2:
        return "中等风险"
    elif score >= 1:
        return "关注"
    else:
        return "正常"
```

## 局限性

### 模型局限

1. **无法预测崩盘具体时间**: 只能预测时间范围
2. **可能出现"假泡沫"**: 有泡沫特征不一定崩盘
3. **参数敏感**: 不同参数可能得出不同结论
4. **事后验证**: 事后看准确，事前预测难

### 使用建议

1. **结合其他指标**: 不要单独使用LPPLS
2. **作为辅助工具**: 识别风险，不是交易信号
3. **设置预警阈值**: 提前提醒，不等崩盘
4. **动态监控**: 持续跟踪信号变化

## 注意事项

1. LPPLS是**风险预警**工具，不是买卖信号
2. 需要结合市场环境和其他指标综合判断
3. 模型假设可能不适用于所有市场
4. 过去准确不代表未来准确
5. **永远设置止损**
