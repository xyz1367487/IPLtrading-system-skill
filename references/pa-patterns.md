# PA价格行为模式详解

## 什么是价格行为(PA)？

Price Action (PA) 是通过裸K线（无指标）识别市场走势的交易方法，核心假设是**价格反映一切信息**。

## 单根K线形态

### 锤子线 (Hammer)
- **特征**: 下影线≥2倍实体，实体小
- **信号**: 底部反转（看多）
- **确认**: 后续阳线确认

### 射击之星 (Shooting Star)
- **特征**: 上影线≥2倍实体，实体小
- **信号**: 顶部反转（看空）
- **确认**: 后续阴线确认

### 十字星 (Doji)
- **特征**: 开盘价≈收盘价
- **信号**: 犹豫/趋势暂停
- **类型**:
  - 长腿十字: 多空分歧大
  - 墓碑十字: 空头占优
  - 蜻蜓十字: 多头占优

### 纺锤线 (Spinning Top)
- **特征**: 上下影线长，实体小
- **信号**: 趋势暂停

## 经典K线组合

### 吞没形态 (Engulfing)

**看涨吞没**:
```
特征:
- 下跌趋势中
- 第一根阴线，第二根阳线
- 阳线实体覆盖阴线实体
- 成交量放大
```

**看跌吞没**:
```
特征:
- 上涨趋势中
- 第一根阳线，第二根阴线
- 阴线实体覆盖阳线实体
- 成交量放大
```

### 孕线 (Harami)

**看涨孕线**:
```
特征:
- 下跌趋势中
- 大阴线后跟小阳线
- 小阳线实体在大阴线实体内
```

**看跌孕线**:
```
特征:
- 上涨趋势中
- 大阳线后跟小阴线
- 小阴线实体在大阳线实体内
```

### 早晨/黄昏之星

**早晨之星 (Morning Star)**:
```
特征（3根K线）:
1. 大阴线 - 下跌趋势
2. 小实体（星线）- 多空平衡
3. 大阳线 - 上涨反转

信号强度: ★★★★★
```

**黄昏之星 (Evening Star)**:
```
特征（3根K线）:
1. 大阳线 - 上涨趋势
2. 小实体（星线）- 多空平衡
3. 大阴线 - 下跌反转

信号强度: ★★★★★
```

### 穿刺/乌云盖顶

**穿刺形态 (Piercing)**:
```
特征:
- 下跌趋势
- 阴线 → 跳空低开高走
- 收盘价超过阴线实体中点

信号强度: ★★★★
```

**乌云盖顶 (Dark Cloud Cover)**:
```
特征:
- 上涨趋势
- 阳线 → 跳空高开低走
- 收盘价超过阳线实体中点

信号强度: ★★★★
```

### 三只乌鸦/三白兵

**三只乌鸦**:
```
特征:
- 连续3根下跌K线
- 收盘价递减
- 常伴成交量放大

信号: 强烈看空
```

**三白兵**:
```
特征:
- 连续3根上涨K线
- 收盘价递增
- 最好小幅高开

信号: 强烈看多
```

## 趋势识别

### 趋势结构

```
上升趋势:
    ↑   ↑   ↑        HH = Higher High
  ↗ ↗   ↗ ↗         HL = Higher Low
↗   ↗ ↗   ↗

下降趋势:
↘   ↘ ↘   ↘         LH = Lower High
  ↘ ↘   ↘   ↘       LL = Lower Low
    ↘   ↘   ↘
```

### 趋势反转信号

**从上涨转为下跌**:
1. 高点降低 (LH出现)
2. 跌破上升趋势线
3. 跌破20日均线
4. 出现看跌形态组合

**从下跌转为上涨**:
1. 高点抬高 (HH出现)
2. 突破下降趋势线
3. 突破20日均线
4. 出现看涨形态组合

## 支撑阻力识别

### 支撑位
- 历史低点
- 均线位置
- 斐波那契回撤位
- 趋势线

### 阻力位
- 历史高点
- 均线位置
- 斐波那契扩展位
- 趋势线

### 支撑阻力转换
```
阻力被突破 → 变成支撑
支撑被跌破 → 变成阻力
```

## 成交量确认

### 量价关系

| 价格 | 成交量 | 信号 |
|------|--------|------|
| 上涨 | 放大 | 强看多 |
| 上涨 | 缩小 | 弱看多（可能背离）|
| 下跌 | 放大 | 强看空 |
| 下跌 | 缩小 | 弱看空（可能背离）|

### 突破确认

```
真突破特征:
- 成交量放大（至少1.5倍）
- 收盘价站稳突破位
- 随后回测不破

假突破特征:
- 成交量未放大
- 长上/下影线
- 快速回落/回升
```

## 使用建议

1. **多周期确认**: 大周期趋势为主，小周期找入场
2. **结合趋势**: 顺势交易成功率更高
3. **等待确认**: 不要预测形态，等待形态完成
4. **量价配合**: 有量配合的形态更可靠
5. **设置止损**: 形态失败立即止损
