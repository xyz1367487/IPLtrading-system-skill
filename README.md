# Trading System Skill

加密货币交易分析系统 - OpenClaw Agent Skill

## 功能

- 获取实时价格（支持 Kraken、CoinGecko、Binance、Coinbase）
- 市场分析（情绪指标、技术面、基本面）
- 制定交易计划（短线/中线策略）
- 计算止盈止损
- 仓位管理建议

## 安装

```bash
# 克隆到 OpenClaw workspace skills 目录
git clone https://github.com/xyz1367487/trading-system-skill.git ~/.qclaw/workspace/skills/trading-system
```

## 使用

在 OpenClaw 中提及交易相关话题即可触发：
- "BTC 现在多少钱"
- "帮我分析一下 ETH"
- "制定一个 BTC 交易计划"
